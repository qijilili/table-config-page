📋 表验收工具 - 配置生成器

填写下方表单，点击「生成配置代码」即可得到可直接粘贴到 Python 脚本中的配置代码。

1. 表名称

例如：dws_zdm.dws_zdm_hive_trade_order_direct_smzdmid
例如：dws_zdm.tmp_dws_zdm_hive_trade_order_direct_smzdmid

2. 老表查询 SQL（OLD_TABLE_SQL）

SELECT 的字段顺序需与下方「字段信息」一致。可用 {OLD_TABLE_NAME} 作为占位符，会自动替换为老表名。

3. 新表查询 SQL（NEW_TABLE_SQL）

可用 {NEW_TABLE_NAME} 作为占位符。新表通常会有额外的 WHERE 条件（如排除美团）。

4. 字段信息（FIELD_INFO）

字段名与类型需与上面 SQL 的 SELECT 字段一一对应。类型：str（维度）、int、float（数值）

5. 可选配置

逗号分隔，如 order_create_date,zdm_id。留空表示用全部 str 字段
0.1 表示 10%，留空表示全量。数据量大时建议填 0.01~0.1
